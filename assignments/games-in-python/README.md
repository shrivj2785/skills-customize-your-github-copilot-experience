
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Create a Hangman game where players guess letters to reveal a hidden word before running out of attempts. This assignment will help you practice string manipulation, loops, conditionals, and random selection in Python.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description
Implement a complete Hangman game that randomly selects a word, accepts user guesses, tracks progress, and determines win/lose conditions.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and display current progress in underscore format (_ _ _)
- Track and display the number of incorrect guesses remaining
- End the game when the word is guessed or all attempts are exhausted
- Display appropriate win or lose messages with the final word revealed
